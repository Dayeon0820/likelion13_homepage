## ✨ 주요 기능

이 홈페이지에서 제공하는 주요 기능입니다.

* **메인 페이지:** [ 13기 활동을 한눈에 볼 수 있는 인트로 및 요약 ]
* **ABOUT US:** [  멋쟁이사자처럼 13기 운영진 소개 ]
* **ACTIVITES 페이지:** [ 13기 교육 일정, 12기 세션 안내, 프로젝트 둘러보기, 활동 갤러리 ]
* **연락처:** [ 문의를 위한 소셜 미디어 링크 ]
* **반응형 웹:** [ 모바일, 태블릿, 데스크탑 환경을 모두 지원 ]
